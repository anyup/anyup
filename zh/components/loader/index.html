# Loader - 加载图标

说明

此组件为一个小动画，目前用在组件的正在加载状态场景。

color 动画活动区域的颜色，只对 mode = flower 模式有效（默认#c7c7c7） String #ffffff - -
size 加载图标的大小 String 34rpx - -
show 是否显示动画 Boolean true false -
